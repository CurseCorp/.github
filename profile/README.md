# Хеллоу, мы - CurseCorp, великая компания 

Мы - команда интеллектуалычей, создающая инструменты и приложения. Наш подход: **Чтобы никто не понял чё мы там понаписали, но работало оно здраво**

## 🛠 Наш стечок
- **Языки**: C++, QML/Qt, C#, Python (Упаси боже)
- **Архитектура**: Строгий MVVM: Мы держим View и ViewModel в разных палатах, чтобы бы не сошли с ума уже мы. Логика не смотрит на интерфейс, а интерфейс не знает, что такое боли бэка.
- **Инструменты**: Git, GitHub Projects, Докер, Опенсервер
